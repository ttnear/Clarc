### New

- **Multi-language support**: The app is now available in Simplified Chinese, Traditional Chinese, Japanese, and Spanish — in addition to English and Korean.
- **Session completion**: Click the circle next to any session to mark it complete. Completed sessions appear dimmed with a strikethrough, and you can hide them with one click to keep your list tidy.
- **Multi-selection in Sessions**: ⌘-click or ⇧-click to select multiple sessions, then right-click to pin, mark complete, or delete them all at once.

### Improved

- **Sessions tab**: The "History" tab is now called "Sessions" across all supported languages.

### Fixed

- Markdown in assistant responses now renders immediately when a text block settles, rather than waiting for the full response to complete.
- Built-in slash commands updated to match the latest Claude Code CLI reference.
- Trailing space below the final response paragraph is now correctly reserved, preventing text from touching the bottom edge.
- Session sidecar fields (pin, completion state) are now preserved when the watcher reloads a session from disk.
